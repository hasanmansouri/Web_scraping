# Web Scraping Project  

This project is a **Python-based web scraper** that extracts company details from a website. It uses **Selenium** to navigate through pages and collect data such as company names, addresses, phone numbers, and websites. The extracted data is saved in a structured CSV file for further analysis.

## Features  
- **Automated Web Scraping**: Collects company details dynamically.  
- **Data Cleaning & Processing**: Formats extracted information for better usability.  
- **CAP Extraction & Grouping**: Extracts postal codes (CAPs) and organizes data accordingly.  
- **Export to CSV & Excel**: Saves data in structured formats.  

## Tools I Used  
- **Python**  
- **Selenium**  
- **Pandas**
